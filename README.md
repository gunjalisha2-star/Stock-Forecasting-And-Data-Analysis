# Stock Market Forecasting and Data Analysis

## Overview

This project focuses on analyzing historical stock market data and predicting future stock prices using Machine Learning techniques. The project includes data cleaning, exploratory data analysis (EDA), visualization, feature engineering, model building, and performance evaluation.

---

## Objectives

* Analyze historical stock market trends.
* Perform data preprocessing and cleaning.
* Visualize stock price movements.
* Build predictive machine learning models.
* Forecast future stock prices.
* Evaluate model performance using statistical metrics.

---

## Technologies Used

* Python
* Pandas

* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Dataset

The dataset contains historical stock market information including:

* Date
* Open Price
* Close Price
* High Price
* Low Price
* Trading Volume

The data was analyzed to identify trends and forecast future stock prices.
